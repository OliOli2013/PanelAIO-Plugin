AIO Panel 14.0.0 — pliki do podmiany na GitHubie

Najważniejsze zmienione pliki:
1. install_softcam_feed_safe.sh
2. legacy_plugin.py
3. changelog.txt
4. SUPERCONFIG_DIRECT_FEED_14.0.0.txt

Do repozytorium można wgrać cały katalog z paczki GitHub albo podmienić powyższe pliki.
Gotowy IPK znajduje się w katalogu release.

Instalacja na tunerze:
opkg install --force-reinstall /tmp/enigma2-plugin-extensions-panelaio_14.0.0_all_DIRECT_FEED.ipk
killall -9 enigma2

Przed ponownym testem można usunąć stare statusy:
rm -f /tmp/PanelAIO/wizard_softcam_*.status /tmp/PanelAIO/wizard_oscam_*.status
rm -f /tmp/PanelAIO/aio_oscam_feed_*.status
rm -rf /tmp/PanelAIO/locks/opkg_softcam.lock
