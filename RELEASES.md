# AIO Panel releases

## 14.0.1 — 2026-07-23

Stabilization release based on the full AIO Panel audit. The primary fix addresses picon installation failures reported on some receivers. It also consolidates the repaired channel-list installer, S4aUpdater lists from the last 12 months excluding Bzyk83, transactional backup/rollback, and the working Super Configurator flow with the direct mynonpublic Softcam feed command and OSCam activation.

## 14.0.0

Major visual and architectural rebuild with adaptive compact/HD/FHD layouts, independent PNG iconography, automatic PL/EN language selection, live receiver health information and the E2 Doctor installer for Python 3 images. The existing action engine remains available through a compatibility adapter to avoid regressions.
