# AIO Panel 14.0.1

AIO Panel to uniwersalny zestaw narzędzi dla tunerów Enigma2 z Pythonem 2 i Pythonem 3.

Repozytorium zawiera wyłącznie aktualne pliki wersji **14.0.1**:

- kod źródłowy wtyczki,
- skrypty instalacyjne i serwisowe,
- autorską paczkę `Picony.zip`,
- jeden aktualny pakiet IPK w katalogu `release`,
- aktualny changelog oraz metadane aktualizacji.

## Instalacja online

```sh
wget -q -O - https://raw.githubusercontent.com/OliOli2013/PanelAIO-Plugin/main/installer.sh | /bin/sh
```

Instalator pobiera wyłącznie aktualny pakiet IPK 14.0.1. Nie pobiera całego repozytorium.

## Budowanie IPK na Linuxie

```sh
./build_ipk.sh
```

Gotowy pakiet zostanie zapisany w katalogu `release/`.
